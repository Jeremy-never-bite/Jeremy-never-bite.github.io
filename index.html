存储系统(Storage Hierachy)(一)

存储体系概述

最小单位 bi
字节 8bits
基本访问操作： 读和写
主存储器，又称内存。CPU可以直接访问，是临时存储器。
辅助存储器，又称外存。如硬盘，CPU要访问外村，先要把外存放入内存。是永久存储器。

存储器分类

介质划分

半导体器件：RAM、ROM
磁性材料：磁盘、磁带
光介质：光盘

存取方式分类

随机存储存储器：主存
顺序存储存储器：磁盘、磁带
相连存储器：按内容访问

读写功能分类

ROM 只读，但是断电后仍存在
RAM 读写

按信息的可保存性

永久保存： 辅存、ROM
非永久保存：RAM

按作用分类

主存： 又称内存
辅存： 又称外存

主存储器

分类

主存分为RAM和ROM
RAM 又分为SRAM和DRAM，SRAM贵快用作Cache，DRAM便宜慢主存
ROM 分为PROM（可编程）、EPROM（紫外线擦除）、E2PROM（电擦除）、flash

存储位元：是MOS静态存储器，相当于底层物理原理
RAM分为DRAM和SRAM

性能指标

存储容量
存储芯片的存储容量 = 存储单元个数 * 每个存储单元的位数
存储速度
存取时间、存储周期、带宽（单位是bps）

层次结构

Cache + 主存 = 内存

存储位元是 六管MOS静态存储器（每一个二进制位的底层物理存储形式）

  1. 使用双稳态触发器表示0和1代码

动态随机读写存储器DRAM(Dynamic Random Access Memory)

存储位元


只有一个MOS管，电容C上有电荷则为1，无电容则为0。
再生：读取电容后信息会被破坏，需要重写
刷新：经过一段时间，信息自己会丢失，需要重写，实质就是“读了再写”

DRAM的刷新方式

对于 阵列为 2048 * 2048 * 4
刷新周期 从上一次刷新阶数到下一次对整个DRAM全部刷新一遍。1个刷新周期要刷2048次
刷新操作 按行来执行内部的读操作。刷新一行所需的时间即是一个存储周期。读取之后如果有电就补电
刷新的行数 单个芯片的单个矩阵的行数
每个矩阵的同一行是同时被刷新的。
单元刷新间隔时间 DRAM允许的最大信息保持时间；一般为2ms（电荷自然消失时间）
刷新方式 集中式刷新、分散式刷新、异步式刷新

集中式刷新

在一个刷新周期内，利用一段固定的时间，以此对DRAM的所有航逐一进行刷新。在这段时间内不允许访存，这段时间被称为死时间。

分散式刷新

把对每一行的刷新分散到各个存储周期中。每个存储周期前一段用于正常的访存操作，后一段用于刷新。
刷了很多遍

异步式刷新

折中的办法，把2ms内分散地把各行刷新一遍。同时又不是集中刷。

DRAM的特点

  1. 使用半导体器件中分布电容上有无电荷来表示0和1
  2. 电源不断电的情况下，信息也会对视，需要不断刷新
  3. 存取速度慢，容量大，价格低
  4. 用于制作内存条
  5. 行列地址分时复用

实例： 注意下面三个点相互独立
